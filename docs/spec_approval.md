이 문서는 모든 HACCP 일지의 화면 출력 및 인쇄 시 적용되는 표준 양식을 정의한다.



1\. 상단 헤더 (Header)

레이아웃: 2단 구성



1행:



\[좌측] 회사 로고 (전사 공통)



\[중앙] 문서 이름 (예: 작업장 위생 점검표)



\[우측] 결재란 (담당 - 검토 - 승인 3칸 표)



2행 (필수 정보):



작성일자: 시스템 자동 생성 (수정 불가)



작성자: 로그인 유저 실명



작성주기: 일간 / 주간 / 월간 등 (양식 설정값)



기타: 필요시 공정 특성에 따른 가변 항목 추가 가능



2\. 하단 푸터 (Footer - 인쇄 전용)

레이아웃: 가로 3칸 표 (페이지 최하단 고정)



기술적 구현 (CSS 지침):



position: fixed; bottom: 0; 스타일을 사용하여 인쇄 시 모든 페이지 하단에 동일하게 출력되도록 설정한다.



브라우저 인쇄 설정의 '머리글/바닥글' 옵션과 별개로, 시스템이 생성하는 공식 문서 바닥글로 작동한다.



내용:



\[중앙/중앙정렬]: 페이지 번호 (Page X / Y)



\[우측/오른쪽정렬]: 개정 번호 (Rev.X)

3\. 관리자 제약 사항 (v1.0)

번호 중복 방지: log\_templates 관리 화면에서 문서번호 입력 시, 기존 번호와 중복되면 \*\*경고창(Alert)\*\*을 띄워 실수를 방지한다.



수동 입력: 모든 문서번호와 Rev 번호는 관리자가 직접 입력한 값을 최우선으로 한다.
